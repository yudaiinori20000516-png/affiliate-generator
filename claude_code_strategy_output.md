- 「失敗したくない」という気持ちが強く、始める前にリスクを知りたい
- 成功談より失敗談の方が信頼できると感じている

### 冒頭フック
> 副業を始めて3ヶ月で失敗したことを数えたら10個あった。全部自分でやらかしたやつだ。同じ遠回りをしてほしくないから全部書く。

### 売れる角度
「失敗10個」という数字の明示が読者に「全部読まなきゃ」と思わせる。暴露系のトーンで書くが煽りすぎない。「自分もやりそう」と思わせるリアルさが命。

### 記事構成
```
【ステップ1】前置き：これは後悔ではなく記録
失敗① ツールを7つ入れて何も完成しなかった
失敗② 最初から有料記事を作ろうとした
失敗③ プロンプトを使い回しすぎた
失敗④ SNS投稿に時間をかけすぎた
失敗⑤ アフィリエイトリンクの確認を怠った
失敗⑥ 記事タイトルを適当にした
失敗⑦ 読者を「全員」に設定していた
失敗⑧ 毎日投稿にこだわって質を下げた
失敗⑨ 収益報告系の記事を真似した
失敗⑩ 「自動化」を目的にした
まとめ：失敗を避けるより、失敗のコストを下げる考え方
```

### 入れるべき体験談
- 各失敗に具体的な数字・状況・気づきを入れる（1〜2行ずつ）
- 「これ笑えない」と読者が思える具体性が鍵
